Title: Edit Control
Text: An edit control is a rectangular control window typically used in a dialog box to permit the user to enter and edit text by typing on the keyboard.
Title: Button
Text:  A button is a control the user can click to provide input to an application.
Title: Combo Box
Text: A combo box is a unique type of control, defined by the COMBOBOX class, that combines much of the functionality of a list box and an edit control.
